Pennsylvania Furries

Howdy!

It's been a fun 13 years running the Pennsylvania Furries website--I brought it online in 2005 and ran the forums that previously existed here until the end of 2018.

The reason for taking the forums down is largely because forums have been surpassed by other forms of communiaction such as Telegram, Twitter and Facebook, at least in the furry community. This is apparent by looking at user posts and user comments over the life of the website. It also didn't help that more and more spammers were signing up.

What comes next?

There are still plenty of ways to stay in touch with other Pennsylvania furries and participate in the community:

  • Discord
    • Pennsylvania Furries on Discord
  • Telegram
    • Pennsylvania Furries on Telegram
    • Pittsburgh Furries on Telegram
    • Lehigh Valley Furries on Telegram
  • Facebook
    • Pennsylvania Furries Facebook Group
    • Pittsburgh Furries Facebook Group
    • Pennsylvania Furries Facebook Page
  • Conventions/Events:
    • Anthrocon
    • Western PA Furry Weekend (WPAFW)
  • Other States:
    • The Furst State - The Furst State or TFS is Delaware's regional furry community, though it has members from surrounding states as well such as Pennsylvania and Maryland.
    • Ohio Furry Events - An easy to access chat for all Ohio furries to post and find events hosted all throughout the state!
  • Archived copy of this website with the forums

FAQ

Q: "The page looks pretty nice, what do you use to build it?"
A: I built the page with Skeleton

Q: "What's your hosting setup like?"
A: This website is static HTML and CSS sitting in an S3 bucket in AWS. SSL is provided by AWS CloudFront.

Q: "How can I contribute to this website?"
A: If you spot a typo or something else amiss, the source for this website is in a GitHub repo, feel free to submit a PR.

Q: "What does the timeline of Pennsylvania Furries look like?"
A:
  • March, 2000: Mailing List Created
  • April, 2003: LiveJournal Group Created
  • July, 2011: Facebook Page Created
  • July, 2012: Facebook Group Created
  • December, 2012: Mailing List Closed Down
  • November, 2015: Forums Brought Online
  • March, 2016: Telegram Group Created
  • December, 2018 : Discord Server Created
  • December 31st, 2018: Forums Closed Down
